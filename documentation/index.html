Getting Started

Thank you for buying Pleasure and supporting Teamfox.

Pleasure is an admin dashboard theme build on Bootstrap. You can use it for your administrator pages. It is responsive, functional and colorful. Our promise is that Pleasure will be constantly kept up to date.

If you want to get news about updates or new themes, please sign up your email address at teamfox.co (we wont spam).

It is very simple to use or modify. In root folder, you will find build folder contains source of javascript and less files.

Lets look at folders to be much more familiarized with Pleasure.

Layout Configuration

Here is basic wireframe of Pleasure Dashboard.

PLEASURE offers some layout settings such material rippled buttons and auto scroll when menu is open. These settings will be valid, if you add classes to the related html markup.

var Layout = {
  settings: {
    rippledButtons: true,
    autoScrollWhenMenuIsActive: false
  }
}

These layout setting also are reachable in assets/admin1/js/layout.js (compiled from build/admin1/js/layout.js).

# Using Colors in HTML Elements

It is easy to use colors in any html element by adding color name to the class attribute of element. For example;

<p class="text-red">
  Sample text
</p>

This code will produce; Sample Text

Or using colors in widgets is similar.

<div class="card card-red">
  ..
</div><!--.card-->

# Getting Colors via Javascript

Sometimes you need to change colors via javascript. You can access defined colors like that

$(document).ready(function () {
  var getColor = Pleasure.colors.red;
  $('.selector').css('color', getColor);
});
